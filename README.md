# Memories Book - Frontend

This is the frontend application for **Memories Book**, a full-stack MERN (MongoDB, Express, React, Node.js) project. It allows users to create, view, edit, search, and delete their favorite memories with a modern and responsive user interface.

## 🚀 Features

- **Responsive & Modern UI**: Built with a visually rich design, including inner shadows, vibrant components, and responsive typography.
- **Interactive Memory Cards**: Beautifully styled cards with dynamic hover effects, smooth transitions, and description truncation for a polished look.
- **Advanced Filtering & Search**: Users can quickly find specific memories by searching via text query or filtering by specific years.
- **Seamless Creation & Editing Forms**: Easy-to-use forms with built-in date selection and field validation.
- **Real-time Backend Integration**: Dynamically fetches and updates data via RESTful APIs connected to the Express backend.

## 🛠️ Technologies Used

- **React.js**: Core frontend library for building component-based interfaces.
- **Vanilla CSS**: Used for crafting a custom, tailored UI with modern aesthetic touches without relying on heavy frontend UI frameworks.
- **JavaScript (ES6+)**: Functional components, hooks, and modern frontend logic.

## 📦 Getting Started

### Prerequisites

You need Node.js and npm installed on your machine.
Ensure the backend server is also set up and running locally to fetch and interact with the database.

### Installation

1. Install the necessary dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

The application will now run in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will automatically reload when you make changes.

### Building for Production

To build the app for production to the `build` folder:
```bash
npm run build
```
It correctly bundles React in production mode and optimizes the build for the best performance.
